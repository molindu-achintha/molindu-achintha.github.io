Diabetic Retinopathy Detection

An ensemble deep learning classifier combining InceptionNet and ResNet for multi-class diabetic retinopathy detection from fundus images, with SMOTE for class imbalance handling.

Project Overview

This project develops an ensemble classifier combining InceptionNet and ResNet outputs to detect and grade diabetic retinopathy from retinal fundus photographs. Uses SMOTE to address class imbalance and comprehensive data augmentation for better generalization.

Problem Statement

Diabetic Retinopathy (DR) is a leading cause of blindness. Early detection through screening can prevent up to 95% of vision loss. Over 400 million people worldwide have diabetes, and ~1 in 3 will develop DR.

Manual grading by ophthalmologists is time-consuming and subject to inter-observer variability. Cannot scale to meet global demand.

Class Imbalance

Most patients are healthy (Grade 0), while severe cases (Grades 3-4) are rare but critical. SMOTE generates synthetic minority samples to balance training.

Model Architecture

InceptionNet V3

Innovation: Inception modules with parallel convolutions (1x1, 3x3, 5x5)
Strength: Multi-scale feature extraction
Params: ~23M

ResNet50

Innovation: Skip connections for deep networks (50 layers)
Strength: Reduced vanishing gradients
Params: ~25M

Ensemble Strategy

Both models run parallel inference, output softmax probabilities are averaged, and argmax gives final prediction. Diversity between architectures improves robustness.

SMOTE Balancing

Synthetic Minority Oversampling

For each minority class sample, SMOTE finds k-nearest neighbors in feature space, creates synthetic samples by interpolation. Minority classes (Grades 3, 4) upsampled to match Grade 0 count.
