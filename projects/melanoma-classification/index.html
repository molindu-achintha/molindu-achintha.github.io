SIIM-ISIC Melanoma Classification

Transfer learning with modified EfficientNet B7 architecture for melanoma detection from dermoscopic images, featuring comprehensive DICOM preprocessing pipeline with CLAHE enhancement.

Project Overview

This project implements a modified EfficientNet B7 architecture with custom classification head for binary melanoma detection from dermoscopic skin lesion images. The solution was developed for the SIIM-ISIC Melanoma Classification Kaggle competition, focusing on early detection of the deadliest form of skin cancer.

Problem Statement

Melanoma is the deadliest form of skin cancer, responsible for 75% of skin cancer deaths. Early detection dramatically improves survival rates - 5-year survival exceeds 99% when caught early, but drops below 25% for late-stage diagnosis.

DICOM Preprocessing Pipeline

Medical DICOM images require specialized preprocessing before feeding to neural networks.

1. DICOM Loading

Parse DICOM files using Pydicom library, extract pixel arrays and metadata

2. Grayscale Conversion

Convert RGB dermoscopic images to grayscale for single-channel processing

3. Gaussian Denoising

Apply Gaussian blur to reduce image noise while preserving lesion boundaries

4. CLAHE Enhancement

Contrast Limited Adaptive Histogram Equalization to enhance lesion visibility

5. Lesion Segmentation

Segment skin lesion from background using thresholding and morphological operations

6. Resize & Normalize

Resize to 224Ã—224, normalize using ImageNet mean/std for transfer learning

CLAHE Details

CLAHE (Contrast Limited Adaptive Histogram Equalization) divides the image into small tiles, applies histogram equalization to each tile with contrast limiting, then stitches them using bilinear interpolation. This enhances local contrast without amplifying noise.

Model Architecture

EfficientNet B7 Base

EfficientNet uses compound scaling to balance network depth, width, and resolution. B7 is the largest variant with:

  • 66M parameters - larger capacity for complex patterns
  • 600Ã—600 input resolution (we use 224Ã—224 for efficiency)
  • MBConv blocks with squeeze-and-excitation attention
  • Swish activation for smoother gradients

Transfer Learning Strategy

  • ImageNet weights: Start with pre-trained EfficientNet B7
  • Freeze early layers: Keep low-level feature extractors frozen
  • Fine-tune later layers: Train last 50 layers with lower learning rate
  • Custom head: Train classification head from scratch
